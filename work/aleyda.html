The problem

There are several options to organize your work, tasks and video calls; all of them separately, one for tasks, one for video calls and one for taking notes on video calls. This does not make sense, now with the new wave of remote work, tools are needed to simplify the workflow of the teams; this problem is serious because it can cause burnout in some employees and even lower productivity.

The solution

The solution was to design a webapp that would allow you to make video calls, create and organize projects, and write notes about your meetings; all this in a single platform, we didn't want to overload the design with links, buttons and that the user would feel stressed by all the functionalities of the webapp so we decided to use very sober colors and a very minimalist design.

Conclusion and learnings

It was a difficult project to design because the client's requirements were to be extremely easy to use but not to leave aside the main features, we learned a lot because managing this project we had to use more than five tools and this helped us to realize which were the points we wanted to highlight, to see which features added the most value.
